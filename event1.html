Festifal Kopi

 

 Hanan

   

&June 22,2017

           BANDUNG-Gubernur Jabar Ahmad Heryawan, membuka secara resmi Festival Kopi Java (Jawa Barat), Jum'at (20/11), bertempat di Trans Studio Mall (TSM) Jl. Gatot Subroto Bandung. Dalam kesempatan itu, Gubernur mengajak semua pihak terutama para pengelola kedai kopi atau cafe untuk menggunakan kopi asli asal Jawa Barat. Karena menurutnya, Kopi Java asal Jawa Barat sudah terkenal ke mancanegara dan bahkan selalu mendapai penilaian terbaik dalam setiap festival tingkat internasional
